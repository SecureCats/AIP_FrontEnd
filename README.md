## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
生成的`dist/`文件夹可能有些奇怪，这是为了适应django的结构而配置的。

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
